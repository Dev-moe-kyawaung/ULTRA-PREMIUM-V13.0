## 🎨 V13.0 Design Features

### **Holographic Futuristic Theme**
- **Color Scheme**: Neon Cyan (#00ffff), Magenta (#ff00ff), Electric Yellow (#ffff00)
- **Visual Style**: Cyberpunk holographic with circuit board aesthetics
- **Geometric Shapes**: Hexagons, diamonds, and angular clip-paths
- **Effects**: Scan lines, grid overlays, floating geometric shapes

### **Unique Components**

#### 1. **Hero Section V13**
- Mouse-follow light effect
- Holographic frame with orbiting tech icons
- Rotating role display with custom animations
- Data stream effects
- Corner brackets UI
- Clipped button shapes

#### 2. **About Section V13**
- Hexagonal grid background
- Tab-based content (Journey / Expertise / Philosophy)
- Vertical timeline with colored nodes
- Animated progress bars
- Philosophy cards with glowing borders

#### 3. **Skills Section V13**
- Hexagonal skill cards
- Circuit board SVG background
- Circular progress indicators inside hexagons
- Tech stack summary cards
- Hover scale and glow effects

#### 4. **Projects & Contact**
- Uses Projects3D and Contact3D from V12 (compatible)
- Can be customized with V13 styling

## 🚀 Installation

```bash
# Clone repository
git clone https://github.com/moekyawaung13721/portfolio-v13.git
cd portfolio-v13

# Install dependencies
npm install

# Start development server
npm run dev



## 📦 Build & Deploy

```bash
# Build for production
npm run build

# Preview build
npm run preview

# Deploy to GitHub Pages
npm run deploy:github
```

## 🎯 Key Differences from V12

| Feature | V12 (3D Glassmorphism) | V13 (Holographic) |
|---------|------------------------|-------------------|
| **Visual Style** | Glassy, frosted glass | Sharp, angular, holographic |
| **Color Palette** | 5 theme variants | Single neon cyberpunk |
| **Shapes** | Rounded corners, spheres | Hexagons, diamonds, angles |
| **Animation Style** | Smooth 3D rotations | Scan lines, data streams |
| **Background** | Particle systems | Circuit boards, hex grids |
| **Typography** | Smooth gradients | Sharp neon glows |
| **UI Elements** | Round buttons | Clipped angular buttons |

## 🎨 Color Customization

Edit color variables in each component's CSS:

```css
:root {
  --neon-cyan: #00ffff;
  --neon-magenta: #ff00ff;
  --neon-yellow: #ffff00;
  --neon-green: #00ff00;
}
```
