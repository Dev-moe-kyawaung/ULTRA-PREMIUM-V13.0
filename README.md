### Prerequisites
- Node.js 18+
- npm or yarn
- Git

### Quick Start

```bash
# Clone repository
git clone https://github.com/moekyawaung13721/portfolio-v13.git
cd portfolio-v13

# Install dependencies
npm install

# Start development server
npm run dev
