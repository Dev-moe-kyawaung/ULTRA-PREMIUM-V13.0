## 🎨 V13.0 Design Features

### **Holographic Futuristic Theme**
- **Color Scheme**: Neon Cyan (#00ffff), Magenta (#ff00ff), Electric Yellow (#ffff00)
- **Visual Style**: Cyberpunk holographic with circuit board aesthetics
- **Geometric Shapes**: Hexagons, diamonds, and angular clip-paths
- **Effects**: Scan lines, grid overlays, floating geometric shapes

### **Unique Components**

#### 1. **Hero Section V13**
- Mouse-follow light effect
- Holographic frame with orbiting tech icons
- Rotating role display with custom animations
- Data stream effects
- Corner brackets UI
- Clipped button shapes

#### 2. **About Section V13**
- Hexagonal grid background
- Tab-based content (Journey / Expertise / Philosophy)
- Vertical timeline with colored nodes
- Animated progress bars
- Philosophy cards with glowing borders

#### 3. **Skills Section V13**
- Hexagonal skill cards
- Circuit board SVG background
- Circular progress indicators inside hexagons
- Tech stack summary cards
- Hover scale and glow effects

#### 4. **Projects & Contact**
- Uses Projects3D and Contact3D from V12 (compatible)
- Can be customized with V13 styling
